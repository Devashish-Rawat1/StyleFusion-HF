# StyleFusion AI
A Neural Style Transfer application using AdaIN.

## Live Demo
(Your Hugging Face Space Link)

## Features
- Upload content + style images
- Adjustable style strength
- Before/After comparison slider
- Download output image
- Interactive model explanation
- Example gallery

## Model Architecture
- VGG Encoder
- Adaptive Instance Normalization (AdaIN)
- Decoder Network

## 🛠 Tech Stack
- PyTorch
- Gradio
- Hugging Face Spaces

##  Demo Flow
1. Upload content image
2. Upload style image
3. Adjust style strength
4. Click Generate
5. Compare + Download result
